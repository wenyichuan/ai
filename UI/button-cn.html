Button 按钮

按钮用于触发一个操作或事件，适用于提交、跳转、打开弹层、执行业务命令等场景。本页面结合 Ant Design 中文文档，展示按钮类型、用法、API 及主题变量。

何时使用

按钮用于标记一个（或封装一组）操作命令，响应用户点击行为，并触发相应的业务逻辑。一个操作区域建议只出现一个主按钮。

  • 🔵 主按钮：用于主要行动点。
  • ⚪ 默认按钮：用于没有主次之分的一组行动点。
  • ➕ 虚线按钮：常用于添加或辅助操作。
  • 🔤 文本按钮：用于最次级行动点。
  • 🔗 链接按钮：用于导航到其他页面或位置。

Ant Design Button 还支持危险、高亮、加载、幽灵、Block、渐变、自定义禁用样式等多种状态与风格。

基础类型

<button class="btn primary">主按钮</button>
<button class="btn default">默认按钮</button>
<button class="btn dashed">虚线按钮</button>
<button class="btn text">文本按钮</button>
<button class="btn link">链接按钮</button>

颜色与变体

通过 type、color、variant 等属性控制按钮风格。推荐主按钮用于主要操作，危险按钮用于危险操作，其他按钮用于补充操作。

蓝色描边按钮可使用类似 .btn.outline 的样式类。

按钮图标

通过 icon 属性可以向按钮中添加图标，增强语义与视觉提示。

<button class="btn primary">✨ 主按钮</button>

图标位置

通过 iconPlacement 可设置图标在按钮中的起始或结束位置。

<button class="btn primary">✨ Start</button>
<button class="btn primary" style="flex-direction:row-reverse">End ✨</button>

按钮尺寸

按钮尺寸分为三种：大按钮 44px 高度，中等按钮 36px 高度，小按钮 28px 高度，分别适用于不同场景。

<button class="btn primary large">大按钮</button>
<button class="btn primary medium">中等按钮</button>
<button class="btn primary small">小按钮</button>

不可用状态

添加 disabled 属性后按钮会变为不可操作状态，同时样式也会变更。

<button class="btn disabled" disabled>不可用</button>

加载中状态

按钮支持 loading 状态，用于异步操作等待反馈，避免重复触发。

<button class="btn loading">加载中</button>

多个按钮组合

在操作区域内建议使用 1 个主按钮 + n 个次按钮。超过 3 个操作时可考虑使用 Dropdown 汇总更多操作。

幽灵按钮

幽灵按钮适合放在色块背景上，背景透明，边框和文字更显轻盈。

危险按钮

危险按钮用于删除、移除、重置等风险操作，建议搭配二次确认。

Block 按钮

Block 按钮会将宽度铺满父容器，适用于横幅 CTA 或表单提交。

渐变按钮

渐变按钮用于品牌类或营销类场景，可以提升视觉表现力。

自定义按钮

Button 支持 autoInsertSpace、classNames、styles 等属性，用于精细化控制按钮显示与语义结构。

  • autoInsertSpace：是否在两个汉字之间自动添加空格。
  • classNames：用于自定义内部语义化 DOM 的 class。
  • styles：传入对象或函数定制内联样式。

Semantic DOM

Ant Design Button 的语义结构常见节点包含 root、icon、content，便于样式扩展与主题定制。

  • root：根元素，包含边框、背景、圆角、过渡、光标等。
  • icon：图标元素，控制图标大小与间距。
  • content：文本内容容器，负责排版与对齐。

FAQ

类型和颜色与变体如何选择？

类型本质上是颜色与变体的语法糖，内部映射为颜色和风格。如果同时设置 variant 和 color，优先使用后者。

如何关闭点击波纹效果？

若不需要点击波纹效果，可通过 ConfigProvider 的 wave={{ disabled: true }} 关闭。
