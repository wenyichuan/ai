Typography 排版

Typography 用于展示标题、段落、文本样式与链接内容，适合文案展示、说明页、列表、日志、详情页等场景。

何时使用

当需要展示标题、段落、列表内容或一列基于文本的基础操作时，适合使用 Typography。

  • 📘 文章、博客、说明书等文本展示
  • ✍️ 文案说明页中的标题和段落
  • 📋 配置项、解释说明、字段说明
  • 🔗 超链接、强调文本、代码样式
  • 🛠️ 可复制、可编辑、自动省略的交互型文本

文本与超链接

内置不同样式的文本，支持默认、次级、成功、警告、危险、标记、代码、链接、删除线、下划线等样式。

Default Success Warning Danger mark code

这是一段 超链接文本，同时支持 加粗、斜体、代码样式 和 删除线。

可编辑

通过 editable 属性可以让文本支持在页面内直接编辑，适合表单类场景或轻量级内容修改。

可编辑文本：点击后可进入编辑状态，修改内容后按 Enter 保存。

editable={{ icon: true, tooltip: '编辑', maxLength: 24 }}

可复制

通过 copyable 属性可以为文本增加一键复制能力，常用于代码、链接、字段值等场景。

复制这段内容可以快速复用到你的说明文案或文档中。

copyable={{ text: 'Hello Typography' }}

省略号

当文字过长时，可使用 ellipsis 自动省略，并支持展开/收起，适合卡片、列表、摘要等场景。

ellipsis={{ rows: 2, expandable: true, symbol: '展开' }}

Semantic DOM

Typography 支持语义化结构，便于样式扩展与主题定制。

  • root：根元素，承载排版基础样式
  • actions：操作栏容器，包含复制、编辑、展开等按钮
  • action：单个操作按钮
  • textarea：编辑模式下的输入区域

FAQ

如何选择标题层级？

通常从页面主标题开始，依次使用 H1-H5 来表达层级结构，避免同一页面中层级跳跃过大。

何时使用省略号？

当内容较长、空间受限、且不希望破坏布局时，省略号是较合适的方式。对于主要信息，建议配合展开/收起使用。
